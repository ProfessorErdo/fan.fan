The Problem

The High Cost of Member Attrition

Decathlon faces significant member attrition each year. Customer churn not only results in direct revenue loss but also undermines long-term customer lifetime value (CLV) and brand loyalty.

The challenge is compounded by the complexity of member behavior: purchase patterns, engagement levels, demographic factors, and seasonal variations all influence churn risk in non-linear ways.

  • ✕ Reactive interventions that come too late
  • ✕ One-size-fits-all retention campaigns
  • ✕ Inability to process multi-dimensional behavioral data
  • ✕ No systematic way to identify high-risk segments
  • ✕ Missed opportunities for proactive engagement

The Solution

A Data-Driven Approach

End-to-end machine learning pipeline that transforms member behavioral data into actionable churn predictions

1

Data Segmentation

Separate members by purchase frequency (low vs. high) to capture distinct churn patterns across different engagement levels

2

Feature Engineering

Create derived features capturing recency, frequency, monetary value, engagement, and derived ratios that reveal churn signals

3

Customer Segmentation

K-means clustering identifies natural member groupings based on behavioral similarity, improving model generalization

4

Feature Selection (RFECV)

Recursive Feature Elimination with Cross-Validation identifies the most predictive features, reducing noise and overfitting

5

Hyperparameter Optimization

Hyperopt Bayesian optimization searches hyperparameter space efficiently, finding optimal configurations for XGBoost, LightGBM, CatBoost

6

Class Imbalance Handling

Tomek Links under-sampling cleans decision boundaries, improving model performance on imbalanced churn data

7

Prediction & Intervention

Generate churn probabilities for current members, rank by risk score, and trigger targeted retention campaigns

Implementation

Data Pipeline

Seven-stage processing pipeline from raw data to churn predictions

Step 0: Data Extraction

Query member data from CNDP warehouse including RFM metrics, engagement, demographics, and churn labels

Step 1: Data Preprocessing

Split by frequency (≤2 vs. >2 purchases), filter invalid records (negative values, empty age groups), handle outliers

Step 2: Feature Engineering

Create derived features: recency_minus_freq, lifespan_times_monetary, top1_sports_spending_percentage, lifespan_devidedby_tenure

Step 3: Customer Segmentation

Transform skewed data (PowerTransformer), scale features (MinMaxScaler), find optimal K (KneeLocator), apply K-means clustering

Step 4: Feature Selection

RFECV with LightGBM, 5-fold stratified CV, precision scoring, select top features that maximize predictive power

Step 5: Model Training

Hyperopt Bayesian optimization (200 trials), Tomek Links under-sampling, 5-fold CV, save best model to OSS

Step 6: Churn Prediction

Load trained model, generate churn probabilities for current members, rank by risk, export to CSV for intervention
